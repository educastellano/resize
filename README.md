# resize-images

Resize images in a directory.

## Prerequisites

* [libvips](https://github.com/jcupitt/libvips)

## Install

    npm install resize-images -g

## Usage

    resize-images albums/selfies 300 200

* it creates the folder `album/selfies-small`
